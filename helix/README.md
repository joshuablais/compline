# Compline & Lauds for Helix

Dark and light colorschemes for [Helix](https://helix-editor.com), based on monastic tradition.

## Installation

To install these themes, copy the files to your Helix configuration directory:

```sh
mkdir -p ~/.config/helix/themes
cp compline lauds ~/.config/helix/themes/
```

Then add one of these to your `~/.config/helix/config.toml` file:

```
theme = "compline"
```

Or:

```
theme = "lauds"
```
